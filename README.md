# ci4smarty
Integration Codeigniter 4 and Smarty Template Engine 

## Requirements

- PHP 7.3+, 8.0+
- CodeIgniter 4.0.4+
- Smarty 4.1.0+

## Installation

Installation is best done via Composer. Assuming Composer is installed globally, you may use
the following command: 

    > composer require maheswara/ci4smarty
